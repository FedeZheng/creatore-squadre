Creatore Squadre

Apri index.html con doppio click per usare l'applicazione.
Funziona in locale, senza internet e senza installazione.

I dati vengono salvati nel browser del computer in uso.
Per spostarli su un altro computer usa Esporta CSV dal database,
poi Importa nello stesso programma aperto sull'altro computer.

Importa accetta CSV, JSON e XLSX. Alcuni file XLS esportati come
tabella sono supportati; se un vecchio XLS binario non si apre,
salvalo da Excel come XLSX e importalo di nuovo.

Il pulsante Modello CSV scarica un file con istruzioni nelle righe
che iniziano con #. Puoi lasciarle nel file: l'import le ignora.

La lingua predefinita delle persone e Cinese. Puoi impostare
Solo italiano o Bilingue dalla scheda persona.
