# Creatore Squadre

Applicazione statica HTML/CSS/JavaScript per gestire un database locale di persone e generare squadre bilanciate.

## Uso

Apri `index.html` oppure pubblica il repository con GitHub Pages.

I dati vengono salvati nel browser del dispositivo in uso. Per spostare i dati su un altro computer usa `Esporta CSV` e poi `Importa`.

## Import

Importa accetta CSV, JSON e XLSX. Alcuni file XLS esportati come tabella sono supportati; se un vecchio XLS binario non si apre, salvalo da Excel come XLSX e importalo di nuovo.

Il pulsante `Modello CSV` scarica un file con istruzioni nelle righe che iniziano con `#`. Puoi lasciarle nel file: l'import le ignora.

## Lingue

La lingua predefinita delle persone e Cinese. Puoi impostare Solo italiano o Bilingue dalla scheda persona. Nella creazione squadre puoi attivare `Dividi per lingua`.
